Personal website for @samkab04, built as a simple static site for GitHub Pages.

## Local preview

Open `/home/runner/work/samkab04.github.io/samkab04.github.io/index.html` in a browser.

## GitHub Pages deployment

This repository is set up as a GitHub Pages user site (`samkab04.github.io`), so publishing from the default branch will serve this site at:

https://samkab04.github.io/

If Pages is not enabled yet:
1. Go to repository **Settings** → **Pages**
2. Set **Build and deployment** source to **Deploy from a branch**
3. Choose the default branch and `/ (root)` folder
4. Save and wait for deployment to complete
